# Recipe Book App

Aplikacja do zarządzania przepisami kulinarnymi z wyszukiwarką i interaktywną listą składników.

## Technologie
- Flask (Python)
- PostgreSQL
- HTML
- CSS
- JavaScript

## Funkcje
- Dodawanie przepisów
- Wyświetlanie listy przepisów
- Podgląd szczegółów przepisu
- Wyszukiwarka przepisów po nazwie lub składnikach
- Interaktywna lista składników (checkbox)
- Kopiowanie instrukcji do schowka

## Uruchomienie
1. Zainstaluj zależności:
```bash
pip install -r requirements.txt
```
2. Uruchom aplikację:
```bash
python app.py
```
